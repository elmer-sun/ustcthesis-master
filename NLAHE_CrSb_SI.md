## Materials and Methods

## Crystal synthesis and characterization

Bulk single crystals of CrSb were grown by a flux method as described in detail elsewhere (52). The obtained crystals are of hexagonal rod-like shape with typical dimensions of 0.2 mm × 2 mm along the short and long axes, respectively. Out-of-plane X-ray diffraction and transmission Laue measurements confirmed the single-crystalline nature of the CrSb samples and revealed that the lateral facets correspond to the {01¯10} planes, with the long axis parallel to the c−axis.

## Microstructured device fabrication using FIB

Crystalline lamellas of CrSb for Hall bar device fabrication were prepared using an FEI Helios G4 UX Dual Beam FIB system. Thin lamellas of dimensions 20 $\mu \mathrm { m } \times 1 0 ~ \mu \mathrm { m } \times 2 ~ \mu \mathrm { m }$ were FIB-cut and extracted from the same piece of bulk CrSb single crystal along different crystallographic directions with an alignment accuracy of $0 . 5 ^ { \circ }$ to the respective crystallographic planes. The lamellae were then transferred to a copper grid using a platinum micromanipulator to perform thinning of the lamella and to remove an amorphous layer resulting from the FIB-cutting process using a low-voltage beam shower. Six-terminal Hall bars of dimensions $1 5 \mu \mathrm { m } \times 8 \mu \mathrm { m }$ were prepared on $\mathrm { S i O _ { 2 } / S i }$ wafers of size 1 cm × 1 cm followed by photolithography and Ti(5 nm)/Au(50 nm) deposition using an e-beam evaporator. Each lamella was welded to the gold electrodes using Pt deposition with the FIB to prepare the final devices.

## Electric transport measurements

All electric transport measurements were performed using a Quantum Design PPMS 6000 system using a four-probe contact geometry. The Hall bar devices were wire-bonded to the chip holder using 25 µm thick aluminum wire. An a.c. bias current of $I = I _ { 0 } \sin ( \omega t )$ with a frequency of $f \ : = \ : \omega / 2 \pi \ : = \ : 1 9 . 3 7 5$ Hz was applied by using an a.c. current source (6221A, Keithley). Unless otherwise noted, an amplitude of $I _ { 0 } = 0 . 6 \mathrm { m A }$ was used. Three lock-in amplifiers (SRS 830, Stanford Research Systems), which are phase matched to the bias current output, were used to simultaneously record nonlinear longitudinal voltages $V _ { \alpha \alpha } ^ { 1 \omega } , V _ { \alpha \alpha \alpha } ^ { 2 \omega } , V _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega } .$ Similarly, nonlinear transverse voltages $V _ { \beta \alpha } ^ { 1 \omega } , V _ { \beta \alpha \alpha } ^ { 2 \omega }$ , and $V _ { \beta \alpha \alpha \alpha } ^ { 3 \omega }$ were recorded simultaneously in a separate experimental run.

## Angle-resolved photoelectron spectroscopy measurements

Conventional ARPES measurements were performed at the beamline UE112 $\mathrm { P G M } { - } 2 \mathrm { b } { - } 1 ^ { 2 }$ of BESSY (Berlin Electron Storage Ring Society for Synchrotron Radiation) synchrotron. The energy and angular resolutions were set to ≈ 20 meV and 0.3◦, respectively, and the temperature was set $\mathrm { t o } \approx 2 0$ K. The bulk crystalline samples for all ARPES measurements were cleaved in situ and measured in a vacuum better than $2 \times 1 0 ^ { - 1 0 }$ Torr.

## Density Functional Theory calculation

Density functional theory (DFT) calculations of the electronic structure were performed using the density functional theory framework as implemented in the Vienna ab initio simulation package (56, 67). The projector-augmented wave potential was adopted with the plane-wave energy cutoff set to 520 eV (convergence criteria $1 0 ^ { - 6 } \mathrm { e V } )$ . The exchange-correlation functional of the Perdew–Burke–Ernzerhof type was used (58) with a $1 6 \times 1 6 \times 1 6$ gamma-centered Monkhorst–Pack mesh. We constructed the Hamiltonian of a Wannier tight-binding model using the WANNIER90 interface (59), including the Cr d-and Sb p-orbitals. We then calculated the Berry curvature and quantum metric from this Wannier model using a $2 0 1 \times 2 0 1 \times 1 5 1$ k-mesh.

## 2.1 Expression of conductivity tensors

We have performed an extensive symmetry analysis to determine the properties of the linear and nonlinear electric conductivity tensors of CrSb by considering the magnetic and crystalline symmetries. The ${ l ^ { t h } }$ -order conductivity tensor is defined as:

$$
J _ { \beta } ^ { ( l ) } = l ! \cdot \sigma _ { \beta \alpha _ { 1 } \alpha _ { 2 } \cdots \alpha _ { l } } E _ { \alpha _ { 1 } } \cdot \cdot \cdot E _ { \alpha _ { l } } ,\tag{1}

where $E _ { a _ { i } } ( i = 1 , 2 , . . . , l )$ is the external electric field along the $a _ { i } \cdot$ -direction $( a _ { i } = x , y , z )$ and $J _ { \beta } ^ { ( l ) }$ is the response current along the β-direction $( \beta = x , y , z )$ . Up to third-order conductivity tensors can be derived using the semiclassical theory of transport (13):

$$
\begin{array} { l } { { \displaystyle \sigma _ { b a } = \sigma _ { b a } ^ { \mathrm { D r u d e } } + \sigma _ { b a } ^ { \mathrm { B C } } } } \\ { { \displaystyle ~ = \tau \frac { e ^ { 2 } } { \hbar ^ { 2 } } \sum _ { n } \int _ { \bf k } f _ { n } \partial _ { a } \partial _ { b } \epsilon _ { n } + \tau ^ { 0 } \frac { e ^ { 2 } } { \hbar } \sum _ { n } \int _ { \bf k } f _ { n } \Omega _ { n } ^ { a b } , } } \end{array}\tag{2}
$$

$$
\begin{array} { l } { { \displaystyle \sigma _ { c a b } = \sigma _ { c a b } ^ { \mathrm { D r u d c } } + \sigma _ { c a b } ^ { \mathrm { B C } } + \sigma _ { c a b } ^ { \mathrm { Q M } } } } \\ { { \displaystyle ~ = - \tau ^ { 2 } \frac { e ^ { 3 } } { \hbar ^ { 3 } } \sum _ { n } \int _ { \bf k } f _ { n } \partial _ { a } \partial _ { b } \partial _ { c } \epsilon _ { n } } } \\ { { \displaystyle ~ + \tau \frac { e ^ { 3 } } { \hbar ^ { 2 } } \sum _ { n } \int _ { \bf k } f _ { n } \frac { 1 } { 2 } \left( \partial _ { a } \Omega _ { n } ^ { b c } + \partial _ { b } \Omega _ { n } ^ { a c } \right) } } \\ { { \displaystyle ~ - \tau ^ { 0 } \frac { e ^ { 3 } } { \hbar } \sum _ { n } \int _ { \bf k } f _ { n } \left( 2 \partial _ { c } G _ { n } ^ { a b } - \frac { 1 } { 2 } \left( \partial _ { a } G _ { n } ^ { b c } + \partial _ { b } G _ { n } ^ { a c } \right) \right) , } } \end{array}\tag{3}

$$

where $\bar { n } \neq n$

For the first-order conductivity tensor in Eq. (2), the first term is the ordinary Drude contribution, which is proportional to the relaxation time τ , and the second term corresponds to the Berry curvature monopole (BCM) independent of $\tau .$

For the second-order conductivity tensor in Eq. (3), the corresponding terms are the secondorder Drude term $( \propto \tau ^ { 2 } )$ , the Berry curvature dipole (BCD) term $( \propto \tau )$ , and the quantum metric dipole (QMD) term $( \propto \tau ^ { 0 } )$ ), respectively.

For the third-order conductivity tensor in Eq. (4), these terms are the third-order Drude term $( \propto \tau ^ { 3 }$ , the Berry curvature quadrupole (BCQ) term $( \propto \tau ^ { 2 } )$ , the quantum metric quadrupole (QMQ) term (∝ τ ), and the AIC term $( \propto \tau ^ { 0 } )$ . Similar to second-order conductivity tensors, BCQ and AIC only have the transverse contribution, while QMQ has both transverse and longitudinal contributions.

## 2.2 Symmetry constraints on conductivity tensors

## 2.2.1 Intrinsic symmetry constraints on conductivity tensors

The intrinsic symmetry constraints on conductivity tensors originate from the fundamental properties of quantum geometric quantities. The quantum metric tensor $G ^ { a b }$ and Berry curvature tensor $\Omega ^ { a b }$ represent the real symmetric and imaginary antisymmetric components of the quantum geometric tensor, respectively. Consequently, $G ^ { a \dot { b } }$ exhibits symmetry under index permutation $( G ^ { a b } = G ^ { b a } )$ , while $\Omega ^ { a b }$ is antisymmetric $( \Omega ^ { a b } = - \Omega ^ { b a } )$ . Furthermore, the multipole moments (e.g., BCD, BCQ) involved in the nonlinear conductivities are formulated as momentum-space derivatives of these fundamental tensors. The indices associated with these derivatives are symmetric $( \partial _ { a } \partial _ { b } = \partial _ { b } \partial _ { a } )$ . These intrinsic symmetry relations significantly constrain the independent components of the conductivity tensors, thus promote the efficiency of their theoretical computation and analysis.

## 2.2.2 Magnetic point group constraints on conductivity tensors

There are three typical elements in the magnetic point group (MPG): inversion $( { \mathcal { P } } )$ , time reversal (T ) and proper rotation (R). We consider these three types of operations on three basic

physical quantities: $\partial _ { \mathbf { k } } , \Omega ^ { a b } ( \mathbf { k } ) , G ^ { a b } ( \mathbf { k } )$ . For $\mathcal { P } , \partial _ { \mathbf { k } }$ and k change signs, and $\Omega ^ { a b }$ and $G ^ { a b }$ keep unchanged. For R, $\partial _ { \mathbf { k } }$ and k transform as vectors, and $\Omega ^ { a b }$ and $G ^ { a b }$ transform as tensors. That is:

$$
\{ \begin{array} { l l } { { \qquad \partial _ { \mathbf { k } }  \partial _ { R \mathbf { k } } } } \\ { { \Omega ^ { a b } ( \mathbf { k } )  \Omega ^ { a ^ { \prime } b ^ { \prime } } ( \mathbf { k } ) = R _ { a ^ { \prime } a } R _ { b ^ { \prime } b } \Omega ^ { a b } ( R ^ { - 1 } \mathbf { k } ) } } \\ { { { \cal G } ^ { a b } ( \mathbf { k } )  { \cal G } ^ { a ^ { \prime } b ^ { \prime } } ( \mathbf { k } ) = R _ { a ^ { \prime } a } R _ { b ^ { \prime } b } { \cal G } ^ { a b } ( R ^ { - 1 } \mathbf { k } ) } } \end{array} \tag{7}
$$

## 4.3 Analysis of the third-order longitudinal response

Our symmetry analysis, presented in Sec. 2, suggests that the $V _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega }$ could arise from either Drude scattering or a QMQ. To understand their specific contributions to $V _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega }$ , we conduct a scaling law analysis and focus on device $' X Z ^ { \prime }$ and $' Y Z ^ { \prime }$ . Drude and QMQ contributions can be identified and determined via the scaling law $E _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega } / ( E _ { \alpha \alpha } ^ { 1 \omega } ) ^ { 3 } = \gamma \sigma _ { \alpha \alpha } ^ { 2 } + \delta$ , where $E _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega }$ corresponds to the electric field of $V _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega } ( ? )$ . Within this scaling law description, QMQ contributions manifest in a vertical intercept δ in the limit $\sigma _ { \alpha \alpha } ^ { 1 \omega }  0$ and Drude scattering contributes a linear relation γ.

As shown in Fig. 5(a) and (b), $E _ { x x x x } ^ { 3 \omega } / ( E _ { x x } ^ { 1 \omega } ) ^ { 3 }$ of device $' X Z ^ { \prime }$ and $E _ { y y y y } ^ { 3 \omega } / ( E _ { y y } ^ { 1 \omega } ) ^ { 3 }$ of device $' Y Z ^ { \prime }$ can be fitted using this scaling law. The finite vertical intercepts $\delta _ { X Z } = - 3 7 . 5 \pm 0 . 3 \times$ $1 0 ^ { 3 } V ^ { - 2 } \mu \mathrm { m } ^ { 2 }$ and $\delta _ { Y Z } = - 2 \pm 1 \times 1 0 ^ { 3 } V ^ { - 2 } \mu \mathrm { m } ^ { 2 }$ indicate contributions from a QMQ to the longitudinal third-order response, whereas the linear slope $\gamma _ { X Z } = ( 5 . 0 0 \pm 0 . 0 1 ) \times 1 0 ^ { 3 } \mu \mathrm { m } ^ { 4 } \mathrm { s } ^ { 2 } / \mathrm { C } ^ { 2 }$ and $\gamma _ { Y Z } = ( 7 . 2 3 \pm 2 . 0 7 ) \times 1 0 ^ { 3 } \mu \mathrm { m } ^ { 4 } \mathrm { s } ^ { 2 } / \mathrm { C } ^ { 2 }$ accounts for contributions from Drude scattering. Therefore, the scaling law analysis of $V _ { \alpha \alpha \alpha \alpha } ^ { 3 \omega }$ is consistent with our theoretical understanding that both Drude scattering and a QMQ can contribute to the longitudinal third order response of any material, magnetic or not.

## 4.4 Theoretical analysis of QMQ in the electronic band structure

In analogy to the analysis of the Berry curvature quadrupole presented in the main text, we examine the presence of QMQ in the electronic band structure of CrSb as derived from density functional theory (DFT) calculations (see Methods section). ## 6 First and second-nonlinear responses of CrSb devices

We have also recorded first- and second-order longitudinal and transverse voltage responses of devices $^ { \prime } X Z ^ { \prime } , ^ { \prime } Y Z ^ { \prime }$ , and $^ { \prime } Z X ^ { \prime }$ , as shown in Fig. 8. As expected, all devices exhibit a linear longitudinal response $V _ { \beta \alpha } ^ { 1 \omega }$ whose amplitude varies between devices, probably owing to variations in their spatial dimensions (thickness, width, and length). We also detect a transverse first-order voltage in all devices, as seen in Fig. 8(b). This observation is surprising. While Drude scattering always contributes to a longitudinal first-order conductivity, that is, the conventional electric conductivity, our symmetry analysis, presented in Sec. II and summarized in Tab. 3, shows that an anomalous contribution to the transverse first-order conductivity, that is, an anomalous Hall effect in linear order is prohibited by the combined crystalline and magnetic symmetries.

To characterize the nature of $V _ { \beta \alpha } ^ { 1 \omega }$ we have also characterized its response to an external magnetic field B applied along the z−direction. An example of such measurement recorded on device $' X Z ^ { \prime }$ is shown in Fig. 9(a). As can be seen, $V _ { z x } ^ { 1 \omega } ( B )$ contains a dominant contribution $V _ { z x , s y m } ^ { 1 \omega }$ that is independent of and symmetric with respect to the magnetic field, $V _ { z x , s y m } ^ { 1 \omega } ( B ) =$ $V _ { z x , s y m } ^ { 1 \omega } ( - B )$ . It can be obtained from the symmetrization operation $V _ { z x , s y m } ^ { 1 \omega } ( B ) = [ \breve { V } _ { z x } ^ { 1 \omega } ( B ) +$ $V _ { z x } ^ { \mathrm { 1 } \omega } ( - B ) ] / 2$ and is shown in Fig. 9(b). In addition, we also detect a much smaller contribution $V _ { z x , H } ^ { 1 \omega }$ that depends linear on the magnetic field $V _ { z x , H } ^ { 1 \omega } \propto B$ and that is, thus, antisymmetric with respect to B, $V _ { z x , H } ^ { 1 \omega } ( B ) = - V _ { z x , H } ^ { 1 \omega } ( - B )$ . We obtain $V _ { z x , H } ^ { 1 \omega } ( B )$ , which is shown in Fig. 9(c), using the antisymmetrization operation $V _ { z x , H } ^ { 1 \omega } ( B ) = [ V _ { z x } ^ { 1 \omega } ( B ) - V _ { z x } ^ { 1 \omega } ( - B ) ] / 2$ . Hence, $V _ { z x , H } ^ { 1 \omega } ,$ which vanishes at $B = 0$ , can be identified as the conventional Hall effect induced by B. The magnetic-field independence of $V _ { z x , s y m } ^ { 1 \omega }$ suggests that $V _ { z x } ^ { 1 \omega }$ at $B = 0$ arises from anisotropy effects, which dominate its amplitude.

## 7.1 Derivations of Conductivity Tensor with Misalignment

As discussed above, although symmetry analysis predicts $\sigma _ { z x } ^ { ( 1 ) } = 0$ for the XZ cut, it can become nonzero in the presence of misalignment. Specifically, the first-order conductivity of the XZ cut is given by:

$$
\sigma ^ { ( 1 ) } = \binom { \sigma _ { x x } ^ { ( 1 ) } } { 0 } \sigma _ { z z } ^ { ( 1 ) } \biggr ) ,\tag{19}
$$

where $\sigma _ { x x } ^ { ( 1 ) } \neq \sigma _ { z z } ^ { ( 1 ) }$ accounts for the anisotropy. This tensor yields a vanishing transverse signal but a finite longitudinal signal.

However, when misalignment is present—characterized by a small angular deviation from the X (or Z) axis, as illustrated in Fig. 10—the conductivity tensor transforms as:

$$
\begin{array} { r l } & { \sigma _ { \mathrm { m i s } } ^ { ( 1 ) } = C ( \theta _ { \mathrm { E } } ) \sigma ^ { ( 1 ) } C ^ { \dagger } ( \theta _ { \mathrm { E } } ) } \\ & { \qquad = \left( \begin{array} { c c } { \sigma ^ { + } + \sigma ^ { - } \cos ( 2 \theta _ { \mathrm { E } } ) } & { \sigma ^ { - } \sin ( 2 \theta _ { \mathrm { E } } ) } \\ { \sigma ^ { - } \sin ( 2 \theta _ { \mathrm { E } } ) } & { \sigma ^ { + } - \sigma ^ { - } \cos ( 2 \theta _ { \mathrm { E } } ) } \end{array} \right) , } \end{array}\tag{20}

## 8.1 Microscopic origin of Berry curvature from band structure properties

We now analyze the microscopic origin of the significant Berry curvature in CrSb, which arises from specific features in its electronic band structure. In the absence of relativistic spinorbit coupling (SOC), bands are double degenerate throughout the $k _ { z } = 0$ plane, protected by the spin point group symmetry $\{ U ( \pi ) | M _ { z } \}$ , where $U ( \pi )$ represents spin-reversal operation and $M _ { z }$ is the mirror operation perpendicular to the z-direction, as shown in Fig. 11(a). When SOC is introduced, it lifts the degeneracy and opens finite gaps in the $k _ { z } ~ = ~ 0$ plane (Fig. 11(b)). These SOC-induced gaps create hot-spots that generate substantial Berry curvature and quantum metric and can induce significant nonlinear transport signals.

## 8.3 Symmetry-determined distributions of Berry curvature and quantum metric

Here we fully analyze the distribution of BC and QM by group theory. The MPG of CrSb is $6 ^ { \prime } / m ^ { \prime } m m ^ { \prime }$ , with generators: identity operation $E ,$ , inversion $\mathcal { P }$ , three-fold rotation $C _ { 3 z }$ , and mirror combined with time reversal operations $M _ { x } \mathcal { T }$ and $M _ { z } \mathcal { T }$ . These generators combine to

form all symmetry operations in the MPG of CrSb, including mirror operation $M _ { y }$ , and two-fold rotations $C _ { 2 x } T , C _ { 2 y }$ , and $C _ { 2 z } \mathcal { T }$

The symmetry-determined properties of the Berry curvature component $\Omega _ { y }$ (whose secondorder derivative corresponds to the experimentally detected third-order transverse signal) can be derived as follows:

$$
\begin{array} { r l r } & { } & { \mathcal { P } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( - { \bf k } ) , } \\ & { } & { M _ { x } \mathcal { T } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( k _ { x } , - k _ { y } , - k _ { z } ) , } \\ & { } & { M _ { y } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( k _ { x } , - k _ { y } , k _ { z } ) , } \\ & { } & { M _ { z } \mathcal { T } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( - k _ { x } , - k _ { y } , k _ { z } ) , } \\ & { } & { C _ { 2 x } \mathcal { T } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( - k _ { x } , k _ { y } , k _ { z } ) , } \\ & { } & { C _ { 2 y } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( - k _ { x } , k _ { y } , - k _ { z } ) , } \\ & { } & { C _ { 2 z } \mathcal { T } : \Omega _ { y } ( { \bf k } ) = \Omega _ { y } ( k _ { x } , k _ { y } , - k _ { z } ) . } \end{array}
$$

From these symmetry constraints, we can conclude that $\Omega _ { y } ( \mathbf { k } )$ exhibits even parity under inversion in all three momentum components: $\Omega _ { y } ( k _ { x } , k _ { y } , k _ { z } ) = \Omega _ { y } ( - k _ { x } , k _ { y } , k _ { z } ) = \Omega _ { y } ( k _ { x } , - k _ { y } , k _ { z } ) =$ $\Omega _ { y } ( k _ { x } , k _ { y } , - k _ { z } )$ This parity analysis is consistent with our first-principles calculations, as visualized through the distribution of $\Omega _ { y }$ in high-symmetry planes.

Despite the even parity of $\Omega _ { y }$ with respect to individual momentum components, the $C _ { 3 z }$ symmetry guarantees that summation of $\Omega _ { y }$ among symmetry-related k-points will be canceled:

$$
\Omega _ { y } ( \mathbf k ) + \Omega _ { y } ( C _ { 3 z } \mathbf k ) + \Omega _ { y } ( C _ { 3 z } ^ { 2 } \mathbf k ) = 0
$$

resulting in vanishing net Berry curvature and consequently zero first-order Hall conductivity, consistent with our experimental and theoretical conductivity tensor analysis.

Following a similar derivation, we analyze the symmetry properties of the quantum metric component $G _ { x x } ( \mathbf { k } )$ . Unlike the Berry curvature, which is odd under time reversal, the quantum metric is even: $\mathcal { T } G _ { x x } ( \mathbf { k } ) = G _ { x x } ( - \mathbf { k } )$ . Applying the MPG symmetry operations yields:

$$
\begin{array} { r l } & { \qquad \mathcal { P } : G _ { x x } ( { \bf k } ) = G _ { x x } ( - { \bf k } ) , } \\ & { M _ { x } \mathcal { T } : G _ { x x } ( { \bf k } ) = G _ { x x } ( k _ { x } , - k _ { y } , - k _ { z } ) , } \\ & { M _ { y } : G _ { x x } ( { \bf k } ) = G _ { x x } ( k _ { x } , - k _ { y } , k _ { z } ) , } \\ & { M _ { z } \mathcal { T } : G _ { x x } ( { \bf k } ) = G _ { x x } ( - k _ { x } , - k _ { y } , k _ { z } ) , } \\ & { C _ { 2 x } \mathcal { T } : G _ { x x } ( { \bf k } ) = G _ { x x } ( - k _ { x } , k _ { y } , k _ { z } ) , } \\ & { C _ { 2 y } : G _ { x x } ( { \bf k } ) = G _ { x x } ( - k _ { x } , k _ { y } , - k _ { z } ) , } \\ & { C _ { 2 z } \mathcal { T } : G _ { x x } ( { \bf k } ) = G _ { x x } ( k _ { x } , k _ { y } , - k _ { z } ) . } \end{array}

## 10 Multiple wave mixing analysis

## 10.1 Foundations of the Quantum Nonlinear Hall Effect

The uniform time-dependent electric field is a superposition of two distinct frequencies with a common real amplitude vector,

$$
\mathbf { E } ( t ) = \mathrm { R e } \Big [ \mathbf { E } _ { 0 } e ^ { i \omega _ { 1 } t } + \mathbf { E } _ { 0 } e ^ { i \omega _ { 2 } t } \Big ] ,\tag{38}
$$

which isolates sum- and difference-frequency responses.

The semiclassical equations of motion in a uniform electric field are

$$
\dot { \mathbf { r } } = \frac { 1 } { \hbar } \nabla _ { \mathbf { k } } \epsilon ( \mathbf { k } ) + \frac { e } { \hbar } \mathbf { E } ( t ) \times \boldsymbol \Omega ( \mathbf { k } ) ,\tag{39}
$$

$$
\dot { \mathbf { k } } = - \frac { e } { \hbar } \mathbf { E } ( t ) .\tag{40}
$$

The term $\propto \mathbf { E } \times \Omega$ is the anomalous velocity, responsible for intrinsic Hall effects.

The distribution function f (k, t) satisfies the Boltzmann equation

$$
\frac { \partial f } { \partial t } - \frac { e } { \hbar } { \bf { E } } ( t ) \cdot \nabla _ { \bf { k } } f = - \frac { f - f _ { 0 } } { \tau } ,\tag{41}
$$

with equilibrium Fermi-Dirac distribution $f _ { 0 }$ and constant relaxation time $\tau .$ . Expanding in powers of the field, $\begin{array} { r } { f = f _ { 0 } + \sum _ { n = 1 } ^ { \infty } f _ { n } } \end{array}$ with $f _ { n } \propto ( { \bf E } _ { 0 } ) ^ { n }$ , gives

$$
\left( \partial _ { t } + \frac { 1 } { \tau } \right) f _ { n + 1 } = \frac { e } { \hbar } \mathbf { E } ( t ) \cdot \nabla _ { \mathbf { k } } f _ { n } .\tag{42}
$$

The charge current density is

$$
{ \bf j } ( t ) = - e \int [ d k ] f ( { \bf k } , t ) { \dot { \bf r } } , \qquad \int [ d k ] \equiv \int \frac { d ^ { d } k } { ( 2 \pi ) ^ { d } } .\tag{43}
$$

Inserting the velocity separates the group-velocity and anomalous-velocity contributions:

$$
{ \bf j } ( t ) = - \frac { e } { \hbar } \int [ d k ] f \nabla _ { \bf k } \epsilon - \frac { e ^ { 2 } } { \hbar } \int [ d k ] f \big ( { \bf E } \times \Omega \big ) .\tag{44}
$$

The second term generates the nonlinear Hall response.

Expanding the current in powers of the field and using the Fourier representation $\mathbf { j } ^ { ( n ) } ( t ) =$ $\textstyle \sum _ { \omega } \tilde { \mathbf { j } } ^ { ( n ) } ( \omega ) e ^ { i \omega t }$ , we define the nonlinear conductivity tensor by

$$
\tilde { j } _ { a } ^ { ( n ) } ( \omega ) = \sigma _ { a ; \alpha _ { 1 } \cdots \alpha _ { n } } ^ { ( n ) } ( \omega ) E _ { 0 , \alpha _ { 1 } } \cdot \cdot \cdot E _ { 0 , \alpha _ { n } } .\tag{45}
$$

In the following we explicitly compute $\sigma ^ { ( 2 ) }$ and $\sigma ^ { ( 3 ) }$ for the Hall current generated by the Berry curvature dipole and quadrupole, i.e., the second term in (44).

## 10.2 First-Order Distribution Function

Setting $n = 0$ in the recurrence relation (42) gives

$$
\left( \partial _ { t } + \frac { 1 } { \tau } \right) f _ { 1 } = \frac { e } { \hbar } { \bf E } ( t ) \cdot \nabla _ { \bf k } f _ { 0 } .
$$

We write the bichromatic field (38) as a sum of positive and negative frequency components,

$$
\mathbf { E } ( t ) = \frac { 1 } { 2 } \sum _ { q = 1 , 2 } \mathbf { E } _ { 0 } \big ( e ^ { i \omega _ { q } t } + e ^ { - i \omega _ { q } t } \big ) , \qquad \mathbf { E } _ { 0 } \in \mathbb { R } ^ { 3 } .\tag{46}
$$

The first-order distribution function is decomposed into the same harmonics:

$$
f _ { 1 } ( { \bf k } , t ) = \sum _ { p = 1 , 2 } \left( F _ { p } ( { \bf k } ) e ^ { i \omega _ { p } t } + F _ { p } ^ { * } ( { \bf k } ) e ^ { - i \omega _ { p } t } \right) .\tag{47}
$$

Substituting into the Boltzmann equation and comparing coefficients of $e ^ { i \omega _ { p } t }$ yields

$$
( i \omega _ { p } + 1 / \tau ) F _ { p } = \frac { e } { 2 \hbar } { \bf E } _ { 0 } { \cdot } \nabla _ { \bf k } f _ { 0 } ,\tag{48}
$$

so that

$$
F _ { p } ( \mathbf { k } ) = \frac { e } { 2 \hbar } \frac { 1 } { i \omega _ { p } + 1 / \tau } \mathbf { E } _ { 0 } \cdot \nabla _ { \mathbf { k } } f _ { 0 } = \frac { e } { 2 \hbar } \frac { \tau } { 1 + i \omega _ { p } \tau } \mathbf { E } _ { 0 } \cdot \nabla _ { \mathbf { k } } f _ { 0 } .\tag{49}
$$

## 10.3 Second-Order Distribution Function

To compute the third-order current we need the second-order correction $f _ { 2 } ( { \bf k } , t )$ . It satisfies the recurrence relation (42) with $n = 1$ :

$$
\left( \partial _ { t } + \gamma \right) f _ { 2 } = \frac { e } { \hbar } { \bf E } ( t ) \cdot \nabla _ { \bf k } f _ { 1 } , \qquad \gamma \equiv \frac { 1 } { \tau } .\tag{50}
$$

the full second-order distribution function is

$$
f _ { 2 } ( { \bf k } , t ) = \hat { f } _ { 2 } ( 0 ) + 2 \mathrm { R e } \sum _ { \omega > 0 } \hat { f } _ { 2 } ( \omega ) e ^ { i \omega t } ,\tag{51}
$$

where

$$
\hat { f } _ { 2 } ( 2 \omega _ { p } ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \zeta _ { p } \zeta ( 2 \omega _ { p } ) { \mathcal L } f _ { 0 } ,\tag{52}
$$

$$
\hat { f } _ { 2 } ( \omega _ { 1 } + \omega _ { 2 } ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \left( \zeta _ { 1 } + \zeta _ { 2 } \right) \zeta ( \omega _ { 1 } + \omega _ { 2 } ) \mathcal { L } f _ { 0 } ,\tag{53}
$$

$$
\hat { f } _ { 2 } ( \omega _ { 1 } - \omega _ { 2 } ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \left( \zeta _ { 1 } + \bar { \zeta } _ { 2 } \right) \zeta ( \omega _ { 1 } - \omega _ { 2 } ) \mathcal { L } f _ { 0 } ,\tag{54}
$$

$$
\hat { f } _ { 2 } ( 0 ) = \frac { S _ { 0 } } { \gamma } = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \left( \zeta _ { 1 } + \bar { \zeta } _ { 1 } + \zeta _ { 2 } + \bar { \zeta } _ { 2 } \right) \mathcal { L } f _ { 0 } .\tag{55}
$$

Here we defined

$$
{ \mathcal { L } } \equiv ( \mathbf { E } _ { 0 } \cdot \nabla ) ^ { 2 }\tag{56}
$$

$$
\zeta ( \omega ) \equiv \frac { 1 } { 1 + i \omega \tau } , \qquad \bar { \zeta } ( \omega ) = \zeta ( - \omega ) = \frac { 1 } { 1 - i \omega \tau } .\tag{57}
$$

In particular we write $\zeta _ { p } \equiv \zeta ( \omega _ { p } ) , \bar { \zeta } _ { p } \equiv \zeta ( - \omega _ { p } )$

## 10.4 Third-Order Anomalous Hall Response from Berry Curvature Quadrupole

The anomalous-velocity contribution to the third-order current is

$$
j _ { a } ^ { ( 3 ) } ( t ) = - \frac { e ^ { 2 } } { \hbar } \epsilon _ { a b c } \int [ d k ] f _ { 2 } ( { \bf k } , t ) E _ { b } ( t ) \Omega _ { c } ( { \bf k } ) ,\tag{58}
$$

Inserting $f _ { 2 }$ given by (51) and the field by (38) ,the complex amplitude at an output frequency $\omega ^ { \prime }$ becomes

$$
\tilde { j } _ { a } ^ { ( 3 ) } ( \omega ^ { \prime } ) = - \frac { e ^ { 2 } } { 2 \hbar } \epsilon _ { a b c } E _ { 0 , b } \sum _ { \omega , q , s = \pm \atop \omega + s \omega _ { q } = \omega ^ { \prime } } \int [ d k ] \Omega _ { c } ( { \bf k } ) \hat { f } _ { 2 } ( \omega ; { \bf k } ) .\tag{59}
$$

All non-zero amplitudes $\hat { f } _ { 2 } ( \omega )$ are proportional to $\mathcal { L } f _ { 0 } = ( \mathbf { E } _ { 0 } \cdot \nabla ) ^ { 2 } f _ { 0 }$ . Integrating by parts twice yields

$$
\int [ d k ] \Omega _ { c } \left( { \bf E } _ { 0 } \cdot \nabla \right) ^ { 2 } f _ { 0 } = \int [ d k ] f _ { 0 } \left( { \bf E } _ { 0 } \cdot \nabla \right) ^ { 2 } \Omega _ { c } = E _ { 0 , d } E _ { 0 , e } Q _ { c d e } ,\tag{60}
$$

where we have introduced the Berry curvature quadrupole tensor

$$
Q _ { c d e } \equiv \int [ d k ] f _ { 0 } ( { \bf k } ) \partial _ { d } \partial _ { e } \Omega _ { c } ( { \bf k } ) .\tag{61}
$$

$Q _ { c d e }$ is a rank-3 pseudotensor, symmetric in its last two indices, and odd under time-reversal. Consequently the integral in (59) can be written as

$$
\int [ d k ] \Omega _ { c } \hat { f } _ { 2 } ( \omega ) = \hat { C } ( \omega ) E _ { 0 , d } E _ { 0 , e } Q _ { c d e } ,\tag{62}
$$

where the prefactors $\hat { C } ( \omega )$ , obtained from (52)–(55) and using ζ(ω) from (57), are

$$
\hat { C } ( 2 \omega _ { p } ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \zeta _ { p } \zeta ( 2 \omega _ { p } ) ,\tag{63}
$$

$$
\hat { C } ( \omega _ { 1 } + \omega _ { 2 } ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \left( \zeta _ { 1 } + \zeta _ { 2 } \right) \zeta ( \omega _ { 1 } + \omega _ { 2 } ) ,\tag{64}
$$

$$
\hat { C } ( \omega _ { 1 } - \omega _ { 2 } ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } \left( \zeta _ { 1 } + \bar { \zeta } _ { 2 } \right) \zeta \left( \omega _ { 1 } - \omega _ { 2 } \right) ,\tag{65}
$$

$$
\hat { C } ( 0 ) = \frac { e ^ { 2 } } { 4 \hbar ^ { 2 } } \tau ^ { 2 } ( \zeta _ { 1 } + \bar { \zeta } _ { 1 } + \zeta _ { 2 } + \bar { \zeta } _ { 2 } ) ,\tag{66}
$$

with $\hat { C } ( - \omega ) = \hat { C } ( \omega ) ^ { * }$ . Substituting (62) into (59) gives

$$
\tilde { j } _ { a } ^ { ( 3 ) } ( \omega ^ { \prime } ) = - \frac { e ^ { 2 } } { 2 \hbar } \epsilon _ { a b c } E _ { 0 , b } E _ { 0 , d } E _ { 0 , e } Q _ { c d e } \sum _ { \omega , q , s \atop \omega + s \omega _ { q } = \omega ^ { \prime } } \hat { C } ( \omega ) .\tag{67}
$$

It is convenient to introduce the dimensionless frequency factor

$$
K ( \omega ^ { \prime } ) = \frac { 4 \hbar ^ { 2 } } { e ^ { 2 } \tau } \sum _ { \omega , q , s \atop \omega + s \omega _ { q } = \omega ^ { \prime } } \hat { C } ( \omega ) ,\tag{68}
$$

so that the current amplitude takes the compact form

$$
\tilde { j } _ { a } ^ { ( 3 ) } ( \omega ^ { \prime } ) = - \frac { e ^ { 4 } } { 8 { \hbar } ^ { 3 } } \tau \epsilon _ { a b c } E _ { 0 , b } E _ { 0 , d } E _ { 0 , e } Q _ { c d e } { \cal K } ( \omega ^ { \prime } ) .\tag{69}
$$

Comparing (69) with the standard definition $\tilde { j } _ { a } ^ { ( 3 ) } ( \omega ) = \sigma _ { a ; b d e } ^ { ( 3 ) } ( \omega ) E _ { 0 , b } E _ { 0 , d } E _ { 0 , e }$ (45) and recalling that $Q _ { c d e }$ is symmetric in d, e, we immediately identify

$$
\sigma _ { a ; b d e } ^ { ( 3 ) } ( \omega ) = - \frac { e ^ { 4 } } { 8 \hbar ^ { 3 } } \tau \epsilon _ { a b c } Q _ { c d e } { \cal K } ( \omega ) .\tag{70}

The explicit forms of $\kappa ( \omega )$ for all distinct positive output frequencies are obtained by collecting the contributing channels $( \omega , s , q )$ in (68) together with (63)–(66). They read:

$$
K ( 3 \omega _ { p } ) = \tau \zeta _ { p } \zeta ( 2 \omega _ { p } ) , \qquad p = 1 , 2 ,\tag{71}
$$

$$
\begin{array} { r } { K ( 2 \omega _ { 1 } + \omega _ { 2 } ) = \tau \left[ ( \zeta _ { 1 } + \zeta _ { 2 } ) \zeta ( \omega _ { 1 } + \omega _ { 2 } ) + \zeta _ { 1 } \zeta ( 2 \omega _ { 1 } ) \right] , } \end{array}\tag{72}

$$
\begin{array} { r } { K ( \omega _ { 1 } + 2 \omega _ { 2 } ) = \tau \left[ ( \zeta _ { 1 } + \zeta _ { 2 } ) \zeta ( \omega _ { 1 } + \omega _ { 2 } ) + \zeta _ { 2 } \zeta ( 2 \omega _ { 2 } ) \right] , } \end{array}\tag{73}

$$
\begin{array} { r } { K ( 2 \omega _ { 1 } - \omega _ { 2 } ) = \tau \big [ ( \zeta _ { 1 } + \bar { \zeta } _ { 2 } ) \zeta ( \omega _ { 1 } - \omega _ { 2 } ) + \zeta _ { 1 } \zeta ( 2 \omega _ { 1 } ) \big ] , \quad ( \omega _ { 1 } > \omega _ { 2 } ) } \end{array}\tag{74}

\begin{array} { r } { \mathcal { K } ( \omega _ { p } ) = \tau \Big [ \zeta _ { 1 } + \bar { \zeta } _ { 1 } + \zeta _ { 2 } + \bar { \zeta } _ { 2 } + ( \zeta _ { 1 } + \zeta _ { 2 } ) \zeta ( \omega _ { 1 } + \omega _ { 2 } ) } \end{array}

+ \zeta _ { p } \zeta ( 2 \omega _ { p } ) + ( \zeta _ { p } + \bar { \zeta } _ { 3 - p } ) \zeta ( \omega _ { p } - \omega _ { 3 - p } ) \Big ] , \quad p = 1 , 2 .\tag{76}

## 10.5 Combined Inductance and BCQ Response

The external drive current contains only the two fundamentals,

$$
I _ { \mathrm { e x t } } ( t ) = I _ { 0 } ( \cos \omega _ { 1 } t + \cos \omega _ { 2 } t ) , \qquad \tilde { I } _ { \mathrm { e x t } } ( \omega _ { 1 } ) = \tilde { I } _ { \mathrm { e x t } } ( \omega _ { 2 } ) = \frac { I _ { 0 } } { 2 } .
$$

In a general AC circuit, the total complex voltage at the measurement terminals is

$$
\begin{array} { l } { \displaystyle \tilde { V } ( \omega ) = \tilde { I } _ { H } ( \omega ) \big ( R + j \omega L _ { s } \big ) + j \omega M \tilde { I } _ { \mathrm { e x t } } ( \omega ) } \\ { \displaystyle = \tilde { I } _ { H } ( \omega ) R \Big [ 1 + j \omega \frac { L _ { s } } { R } + j \omega \frac { M } { R } \frac { \tilde { I } _ { \mathrm { e x t } } ( \omega ) } { \tilde { I } _ { H } ( \omega ) } \Big ] . } \end{array}\tag{77}
$$

We define their fixed complex ratio

$$
\frac { \tilde { I } _ { \mathrm { e x t } } ( \omega ) } { \tilde { I } _ { H } ( \omega ) } \equiv \beta ( \omega ) .
$$

In the low-frequency limit $\omega _ { p } \tau \ll 1 , \beta ( \omega )$ is approximately real as well.

In this regime the inductive terms combine into an effective inductance:

$$
L _ { \mathrm { e f f } } ( \omega ) = L _ { s } + M \beta ( \omega ) , \qquad \beta ( \omega ) = \left\{ \begin{array} { l l } { \beta , } & { \omega = \pm \omega _ { 1 } , \pm \omega _ { 2 } } \\ { 0 , } & { \mathrm { o t h e r w i s e } . } \end{array} \right.
$$

and the voltage amplitude simplifies to

$$
| \tilde { V } ( \omega _ { p } ) | \propto | \tilde { j } ( \omega _ { p } ) | \sqrt { 1 + \left( \omega _ { p } L _ { \mathrm { e f f } } ( \omega _ { p } ) / R \right) ^ { 2 } } .
$$
